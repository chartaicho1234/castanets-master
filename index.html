カスタネットマスター

高精度リズムトレーニングアプリ

BPM 180 - 4分音符・8分音符・16分音符

🎯 高精度タイミング判定

±25ms以内でパーフェクト判定。プロレベルの精度でリズム感を鍛えます。キャリブレーション機能でデバイスの遅延も自動補正。

📊 詳細な分析機能

タイミングの偏差をリアルタイムで可視化。セグメント別の分析や統計情報で、あなたの弱点を明確に把握できます。

🎵 段階的レベル設計

4分音符から16分音符まで、段階的にレベルアップ。各レベルで休符を含む実践的なリズムパターンを練習できます。

⚙️ キャリブレーション

デバイスやオーディオの遅延を自動検出・補正。どんな環境でも正確なタイミング判定を実現します。

開発・実行方法

  1. リポジトリをクローン: git clone https://github.com/yourusername/castanets-master.git
  2. 依存関係をインストール: npm install
  3. 開発サーバーを起動: npm run dev
  4. ブラウザで http://localhost:8081 にアクセス
  5. モバイルデバイスでテストする場合は Expo Go アプリを使用
